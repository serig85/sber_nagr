4. Создан SQL запрос для получения выборки:
	"SELECT first_name, last_name, Credit_Number, Balance "
                           "FROM (SELECT * FROM client JOIN relation on client.id = relation.client) "

6. В Python получены данные запроса и произведено проектирование скрипта для записи отчёта.
